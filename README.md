## Local Development

For local development with full server capabilities:

```bash
# Start the application
npm start

# Or use the startup scripts
./scripts/Dave.sh    # Linux/macOS
scripts\Dave.bat     # Windows
```

Then open http://localhost:7777/ in your browser.
